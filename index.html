Projects

 

Para is an open source back-end framework which helps you build your project faster by taking care of all the boring backend operations.

Para BaaS is our core product - cloud-based hosting for the Para backend server, with an advanced management console and strong two-factor authentication.

Scoold is a StackOverflow clone written in Java and powered by Para. It's a production-ready Q&A platform, which can be deployed on Heroku or your own infrastructure.

Para Web Console is an open source management console for Para. Connect to any of your servers and manage your objects easily with this powerful web interface.
